Principais Formatações

Negrito / Destaque

Podemos utilizar a tag <b> PARA DEIXAR EM NEGRITO (não semântica)

Podemos utilizar a tag <strong> PARA DEIXAR EM DESTAQUE (semântica)

Italico / Ênfase

Podemos utilizar a tag <i> PARA DEIXAR ITÁLICO (não semântica)

Podemos utilizar a tag <EM> PARA DAR ÊNFASE (semântico)

Texto marcado

Podemos utilizar a tag <MARK> PARA MARCAR

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto excluido para indicar que ele deve ser lido, porém não considerado utilizado a tag <DEL>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois utilizando a tag <INS>

Texto sobreescrito

Para inserir coisas como x20 + 3 = 2 utilizando a tag <SUP>

Texto subescrito

Parar inserir coisas como H20 utilizando a tag <SUB>
